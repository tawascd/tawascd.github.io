Tonight's dances
  1. Crabbit Shona (J ) - This was our warm-up walking dance. https://my.strathspey.org/dd/dance/13259/ 
  2. Flight to Melbourne (R) - This is a square set dance. https://my.strathspey.org/dd/dance/11483/ 
  3. The Lammermuir Hills (S) - This dance by Roy Goldring begins with an all round poussette, and finishes with a 2C Knot. https://my.strathspey.org/dd/dance/3678/ Both these formations needed some practise. We began by revising the strathspey setting step, which goes sideways, because that is what is needed for the all round poussette. Pointers for the poussette: start from the sidelines; start on 'outside' leg; formation is shaped like a diamond; it is not a progression.
  4. Kathy's Fascinator (J) - https://my.strathspey.org/dd/dance/20049/ This dance starts with Set and Rotate, and also finishes with a 2C Knot (but this time it's in fast time!). We first practised the Set and Rotate (but left out the casting bit). 
  5. Orpington Caledonians (R) - We did this last week. https://my.strathspey.org/dd/dance/16523/ 
  6. Miss Eleanor (S) - https://my.strathspey.org/dd/dance/16526/  This included a 3C Bourrel, which we had to practise first. 
  7. John of Bon Accord (R) - This was for the more experienced dancers. It starts with four progressive half-reels, which are tricky and you need your wits about you.  The fourth video where the dancers wear numbered bibs is the best one to watch to figure out what is happening. https://my.strathspey.org/dd/dance/3326/
